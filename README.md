# Native American Crime Novels

A study of Tony Hillerman's Leaphorn & Chee mystery series exploring Navajo culture, landscape, and justice.

## Website

[https://dmccreary.github.io/native-american-crime-novels/](https://dmccreary.github.io/native-american-crime-novels/)

## About

This project explores the themes, characters, and cultural elements in Tony Hillerman's acclaimed detective novels set in the Navajo Nation. The series features Lieutenant Joe Leaphorn and Officer Jim Chee of the Navajo Tribal Police.
